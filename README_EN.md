#### Practicing Sentences

After opening a lesson, the left side shows the sentence list and the right side is the practice panel.

**Practice flow:**

1. **Select a sentence**: Click any sentence on the left — the right panel shows the English, Chinese translation, and phonetic notation
2. **Listen to the model**: Click "🔊 Listen" to hear TTS pronunciation
3. **Start recording**: Click "🎤 Record" and speak into your microphone
4. **Stop recording**: Click "⏹ Stop" when you're done
5. **Review your score**: A pronunciation score and word-by-word feedback appear
6. **Continue**: Click "Next →" to move to the next sentence

> **Tip:** Listen first → silently repeat → then record. This is the most effective approach!

**Right panel buttons:**

| Button | Function |
|--------|----------|
| 🔊 Listen | Play TTS standard American English |
| 🎤 Record | Start microphone recording |
| ⏹ Stop | Stop recording |
| Next → | Jump to the next sentence |

### 💬 Conversation Drills

Simulated real-world scenarios with four-choice questions.

#### How to practice

1. Click "💬 Conversation Drills" in the sidebar
2. Choose a topic from the left panel
3. Read the English scenario (with Chinese hint below)
4. Choose the most natural English response from 4 options
5. Instant feedback:
   - ✅ Correct: Shown in green, with an explanation
   - ❌ Incorrect: The correct answer is revealed with a detailed explanation
6. Click "Next →" to continue

#### Available Topics

Daily chat, restaurant ordering, travel & directions, workplace communication, shopping, job interviews, and more.

## FAQ

**Q: Microphone not working?**

A: Microphone access requires HTTPS. Opening the file directly via `file://` will block mic access in Chrome. Use an HTTPS URL or start a local server:

```bash
# Start a local server with Python
python -m http.server 8080
# Then open http://localhost:8080
```

You can also use the **Live Server** extension in VS Code: right-click `index.html` → Open with Live Server.

---

**Q: No microphone — can I still use the app?**

A: Yes. Without a microphone you can still browse lessons, listen to TTS model pronunciation, complete conversation drills, read learning tips, and view your progress. Only the recording feature requires a mic.

---

**Q: Where is my data stored? Is it private?**

A: All data (completion records, recording count, drill scores) is stored exclusively in **your browser's localStorage**. Nothing is uploaded to any server and no one else can see it.

---

**Q: Will my data transfer to another computer or browser?**

A: No. Progress is stored only in the current browser on the current device. Clearing browser data, switching computers, or switching browsers will reset your progress.

---

**Q: No sound when I click "Listen"?**

A:
1. Check browser volume is not muted
2. Check the browser allows audio playback (click the lock icon in the address bar)
3. TTS requires an internet connection — it won't work offline
4. Try switching to Chrome or Edge

---

**Q: Does it work on mobile?**

A: Yes. SpeakFlow uses responsive design (RWD) and automatically adapts to mobile phones, tablets, and desktops of any screen size. Recommended: Chrome (Android) or Safari (iOS).
