## 功能詳細說明

### 🏠 首頁

進入網站後的第一個畫面。

**可以看到：**
- **學習統計**：今日練習句數、練習天數、獲得徽章數
- **今日課程推薦**：系統推薦一堂適合今天練習的課程
- **本週學習紀錄**：7 天的打卡圖，顯示哪幾天有練習
- **快速練習入口**：點選情境直接跳到對應課程

**操作方式：**
- 點「🚀 開始今日課程」立即進入推薦課程
- 點週曆下方的情境卡片，快速跳到該主題練習

### 📚 課程練習

核心功能，透過情境句子練習口說。

#### 課程列表

左側選單點「📚 課程」後，可看到所有可用課程。

**篩選課程：**
- 點上方分類標籤（全部 / 食 / 衣 / 住 / 行 / 育 / 樂 / 日常 / 職場 / 旅遊 / 社交）
- 每張課程卡片顯示：主題、句子數量、預估時間、難度

**課程卡片圖示說明：**
- 🟢 初級 / 🟡 中級 / 🔴 進階
- ✓（綠色圓點）：此課程已完成過

#### 練習句子

點進課程後，左側顯示所有句子列表，右側為練習面板。

**練習流程：**

1. **選擇句子**：點選左側任一句子，右側顯示英文、中文翻譯與音標
2. **聽示範**：點「🔊 聽示範」按鈕，系統用 TTS 朗讀標準發音
3. **開始錄音**：點「🎙️ 開始錄音」，對著麥克風跟著說
4. **停止錄音**：說完後點「⏹ 停止」
5. **回聽自己**：點「▶ 播放錄音」聽自己的發音
6. **繼續下一句**：點「下一句 →」

> **小技巧：** 先聽示範 → 跟著默念 → 再錄音，效果最好！

**右側面板按鈕說明：**

| 按鈕 | 功能 |
|------|------|
| 🔊 聽示範 | 播放 TTS 標準美式發音 |
| 🎙️ 開始錄音 | 啟動麥克風錄音 |
| ⏹ 停止 | 停止錄音 |
| ▶ 播放錄音 | 回聽剛才的錄音 |
| 下一句 → | 跳到下一個句子 |

**進度條：** 練習區上方顯示「第 X / Y 句」，右側顯示已完成百分比。

**完成課程：** 練完所有句子後，課程自動標記為完成（✓），並在進度頁記錄。

### 💬 對話練習

模擬真實情境的四選一選擇題。

#### 如何練習

1. 左側選單點「💬 對話練習」
2. 系統顯示一個英文情境描述（中文輔助說明）
3. 閱讀題目，從 A、B、C、D 四個選項選出最自然的英文回答
4. 點選後立即顯示：
   - ✅ 答對：顯示綠色，附上解析說明
   - ❌ 答錯：顯示正確答案與詳細解析
5. 點「下一題 →」繼續

#### 選題邏輯

- 題目依分類（食 / 衣 / 住 / 行...）分類
- 每次進入會隨機抽題，不會每次順序相同
- 作答紀錄計入「學習進度」的正確率統計

| 😌 降低焦慮 | 放鬆狀態下學習效率最高 |

## 常見問題

**Q：麥克風無法使用？**

A：需要 HTTPS 環境。若在本機以 `file://` 直接開啟，Chrome 預設會封鎖麥克風。請使用支援 HTTPS 的網址開啟網站，或在本機啟動伺服器：

```bash
# 用 Python 啟動本機伺服器
python -m http.server 8080
# 然後開啟 http://localhost:8080
```

也可安裝 VS Code 的 **Live Server** 擴充套件，右鍵點 `index.html` → Open with Live Server。

---

**Q：沒有麥克風，可以用嗎？**

A：可以。沒有麥克風仍可使用課程瀏覽、聽 TTS 示範發音、對話練習、學習技巧等功能，只有「錄音」功能需要麥克風。

---

**Q：學習資料存在哪裡？會不會被別人看到？**

A：所有資料（完成紀錄、錄音次數、對話成績）僅存在**你自己瀏覽器的 localStorage**，完全不會上傳到任何伺服器，其他人無法看到。

---

**Q：換電腦或換瀏覽器，資料還在嗎？**

A：不會在。學習進度只存在目前這台電腦的這個瀏覽器。清除瀏覽器資料、換電腦、換瀏覽器後會重新開始。

---

**Q：「聽示範」沒有聲音？**

A：
1. 確認瀏覽器音量未靜音
2. 確認瀏覽器允許播放音訊（網址列左側的鎖頭圖示）
3. TTS 示範需要連網，離線狀態下無法使用
4. 換用 Chrome 或 Edge 試試

---

**Q：手機可以用嗎？**

A：可以。網站支援手機瀏覽器，介面會自適應螢幕大小。建議使用 Chrome（Android）或 Safari（iOS）。
